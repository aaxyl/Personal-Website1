Project Ideas

Lowcost Arduino Quadcopter Drone

Overview of the project

This project will involve the construction of a relatively low cost quadcopter drone consisting of a quadcopter frame (has to be sturdy, on top of being lightweight as well), rotors (each fitted with electronic speed controllers), propellers, a battery pack, IMU, an arduino board (w/ or w/o flight controller components soldered on, the latter would involve more steps & electronic components) and a RC controller used for remote control of the quadcopter. Some aerodynamic principles (i.e: mainly thrust) would be taken into consideration when purchasing or making my own propellers, as it is crucial for propelling the drone itself into the air. Programming is also involved in linking the arduino to the flight controller components, especially the RC controller which allows for controlling the quadcopter via remote control.

Motivation for the project

There is a currently ever increasing market for drones, and with that, more and more people who'd want one. However most quality ones come at a hefty cost. As such, this project aspires to rectify the problem of finances by providing a way for ordinary people and enthusiasts alike to be able to build their own drones, specifically the quadcopter variant by using an Arduino board and low-cost but durable materials used in the physical construction of the drone.

Project Description

Tools and Technologies required

The technologies required for this project would consist of various sturdy and lightweight materials used in the physical construction of the quadcopter (i.e: quadcopter frame w/ center mounting plate, x4 arms, x4 motor brackets, rotors each w/ electronic speed controllers, counter-wise brushless motors and propellers), as well as an Arduino board w/ the accompanying Arduino IDE software, a battery pack (most preferably LiPo), an IMU (inertial measurement unit), a RC controller unit and an accompanying remote control unit. Tools required for the construction of the quadcopter itself would involve a construction tookit, a soldering iron & lead, multi-color wires, a multimeter & LED displays for testing purposes.

Skills required

Good knowledge in programming (using the YMFC-AL code is a good point to start off with)

Experience in soldering electronics (if purchasing a stock arduino)

Outcome

If everything goes well, I would have a working, inexpensive, Arduino-powered quadcopter made from scratch.
